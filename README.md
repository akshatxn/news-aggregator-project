## вңЁ What is AI Smart News Sphere?

**AI Smart News Sphere** is a dual-mode intelligent news aggregator that goes beyond simple headlines. It combines advanced **Natural Language Processing (NLP)** with **Generative AI** to cluster, analyze, and surface deep insights from news stories вҖ” so you don't just know *what* happened, you understand *why* it matters.

The project ships in two distinct but complementary forms:

| Mode | Component | Purpose |
|------|-----------|---------|
| рҹ–ҘпёҸ **Web Dashboard** | `news_aggregator.html` | Responsive Thistle-themed UI with Gemini Pro AI analysis |
| рҹҗҚ **Python CLI Engine** | `smart_newssphere.py` | Live data fetching, TF-IDF clustering, terminal reports |

---

## рҹҡҖ Features

### рҹ–ҘпёҸ Web Dashboard вҖ” `news_aggregator.html`

> Powered by **Google Gemini Pro** for rich, multi-layered analysis.

- **рҹӨ– AI-Powered Analysis** вҖ” Generates summaries, sentiment scores, and contextual insights per article
- **рҹ”Қ Root Cause Analysis** вҖ” Explains the *why* behind every story, not just the *what*
- **рҹ“ң Historical Context** вҖ” Connects current events to relevant past occurrences
- **вқ“ Follow-up Questions** вҖ” Auto-generates thought-provoking questions for deeper reading
- **рҹҺЁ Modern Thistle UI** вҖ” Polished purple/white responsive grid layout
- **рҹ”Һ Smart Filtering** вҖ” Filter articles by **Category**, **Location** (USA/India), and **Type**

---

### рҹҗҚ Python CLI Engine вҖ” `smart_newssphere.py`

> A robust ML-backed backend that runs entirely in your terminal.

- **рҹ“Ў Live Data Fetching** вҖ” Pulls real-time articles from **NewsData.io API** and **BBC RSS feeds**
- **рҹ§  TF-IDF Vectorization** вҖ” Converts raw text into machine-readable feature vectors
- **рҹ—ӮпёҸ Agglomerative Clustering** вҖ” Groups semantically related stories into meaningful topic clusters automatically
- **рҹ“Ҡ Console Reporting** вҖ” Structured text summaries and clustering stats printed in your terminal

## рҹ–ҘпёҸ Running the Web Dashboard

The web dashboard requires a local server to avoid browser CORS security blocks.

**Step 1 вҖ” Start a local server:**
```bash
cd path/to/NewsSphere
python -m http.server 8000
```

**Step 2 вҖ” Open in your browser:**
```
http://localhost:8000/news_aggregator.html
```

**Step 3 вҖ” Configure your Gemini API Key:**

Open `news_aggregator.html` in any code editor and find line ~380:
```javascript
const apiKey = "";  // рҹ‘Ҳ Paste your Google Gemini API Key here
```

---

## рҹҗҚ Running the Python CLI Engine

The CLI engine fetches live news and runs ML clustering directly in your terminal.

**Step 1 вҖ” Set your NewsData.io API Key:**

Open `fetcher.py` and set:
```python
NEWSDATA_API_KEY = "your_api_key_here"
```

**Step 2 вҖ” Run the main script:**
```bash
python smart_newssphere.py
```

The script will:
1. рҹ“Ў Fetch live articles from NewsData.io and BBC RSS
2. рҹ§  Run TF-IDF vectorization on article content
3. рҹ—ӮпёҸ Cluster related stories using Agglomerative Clustering
4. рҹ“Ҡ Print grouped topic summaries to your console
